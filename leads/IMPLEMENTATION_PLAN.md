# RetailOS Implementation Plan

## Phase 1

1. Create the React/Vite/TypeScript/Tailwind foundation.
2. Add shadcn/ui-style primitives for buttons, inputs, cards, badges, dialogs, tables, and navigation.
3. Configure Supabase Auth and tenant-aware data access.
4. Build onboarding for first business creation and owner membership.
5. Build dashboard analytics from live invoice and customer tables.
6. Build customer CRM with search, filters, tags, notes, lifetime value, visit count, and purchase history.
7. Build invoice generation with line items, auto invoice number, PDF export, thermal print, A4 print, WhatsApp share, and review QR.
8. Build branding settings for logo URL, brand color, business identity, address, phone, review link, invoice footer, and default printer size.
9. Add PWA manifest and service worker.
10. Add Supabase schema, RLS policies, tenant helpers, audit logs, and storage bucket policy.

## Phase 2

Campaign manager, segmentation, broadcast list generation, and reports.

## Phase 3

WhatsApp API integration and automations.

## Phase 4

Loyalty, advanced analytics, and AI insights.
